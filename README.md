# WaterRPA - 傻瓜式RPA自动化工具

**无需编程，图形化配置和执行自动化任务**

---

## 📋 项目概述

WaterRPA 是基于Python和PySide6开发的**图形化RPA工具**，支持9种自动化操作，支持图像识别、JSON配置导入导出、循环执行等功能。

**Version B 新增**：Ctrl+V 快速粘贴截图、自动压缩、图像历史管理。

### 快速开始

#### 安装
```bash
# 创建虚拟环境并激活（推荐）
python -m venv .venv
.venv\Scripts\activate          # Windows
# 或 PowerShell: .\.venv\Scripts\Activate.ps1

# 安装依赖
pip install -r requirements.txt

# 启动应用
python waterRPA_GUI.py
```

#### 系统要求
- Python 3.10+
- Windows 10/11
- 内存 2GB+

## 💡 功能特性

### 支持的操作类型（9种）

| 操作编码 | 操作名称 | 参数示例 | 应用场景 |
|---------|--------|--------|--------|
| 1.0 | **左键单击** | `button.png` | 点击网页按钮、菜单项等 |
| 2.0 | **左键双击** | `icon.png` | 打开文件、启动应用 |
| 3.0 | **右键单击** | `file.png` | 右键菜单操作 |
| 4.0 | **输入文本** | `Hello World` | 填充表单、搜索框 |
| 5.0 | **等待(秒)** | `2.5` | 等待页面加载、系统响应 |
| 6.0 | **滚轮滑动** | `5` 或 `-5` | 上下滚动网页内容 |
| 7.0 | **系统按键** | `ctrl+c`, `alt+tab` | 复制粘贴、切换窗口 |
| 8.0 | **鼠标悬停** | `target.png` | 鼠标移动到特定位置 |
| 9.0 | **截图保存** | `D:\Screenshots` | 保存屏幕快照、生成报告 |

### 高级特性

#### �️ 智能截图粘贴功能 (Version B新增)

WaterRPA 现在支持四种方式输入图像参数，其中**Ctrl+V 粘贴**是最便捷的新功能：

**四种输入方式：**
1. **Ctrl+V 快速粘贴** (推荐) - 用 `Win+Shift+S` 截图，在参数框按 Ctrl+V 即可
2. **拖拽输入** - 直接从文件管理器拖拽图片文件到参数框
3. **文件浏览** (原有) - 点击【选择图片】按钮浏览选择
4. **路径输入** (原有) - 直接输入图片文件的完整路径

**核心功能说明：**

| 功能 | 说明 |
|------|------|
| **自动压缩** | 图片大小超过 1MB 时自动提示压缩，压缩到 1920px 宽度，通常节省70-90%空间 |
| **智能识别** | 识别图像操作类型（1.0/2.0/3.0/8.0），自动设置正确的操作代码 |
| **历史管理** | 所有粘贴的图片自动保存到 `template/` 文件夹，点击 📋 按钮查看历史重用 |
| **文件防覆盖** | 同秒内相同图片自动计数 (screenshot_YYYYMMDD_HHMMSS_1.png 等) |
| **无损保存** | PNG 格式无损质量，保留图像识别所需的清晰度 |

**使用示例（最快的工作流）：**
```
1. 用 Win+Shift+S 截图并自动复制
2. 点击参数输入框
3. 按 Ctrl+V 粘贴
4. 选择是否压缩，点击【保存】
5. 完成！文件自动保存到 template/ 并显示在输入框
```

#### �🔄 重试机制
- **1** = 尝试一次后放弃
- **-1** = 无限重试（直到找到或超时）
- **n** = 重试n次（n > 1）
- **超时保护** = 默认60秒超时，防止卡死

#### 🔁 执行模式
- **执行一次** = 顺序执行完所有任务后停止
- **循环执行** = 不断重复执行任务序列（按Ctrl+C停止）

#### ⚙️ 运行选项
- **运行时最小化** = 任务运行时自动最小化窗口，完成后自动还原

## 📖 快速使用

### 基本步骤
1. **启动应用** - 双击 `waterRPA_GUI.py`
2. **添加任务** - 点击【+新增】，选择操作类型，输入参数
   - **图像操作**：推荐用 Win+Shift+S 截图后 Ctrl+V 粘贴，或浏览文件
   - **文本操作**：直接输入内容
   - **等待**：输入秒数
   - **快捷键**：输入如 `ctrl+c` 的组合
3. **保存配置** - 点击【保存配置】为JSON文件（可重复使用）
4. **运行任务** - 选择执行模式，点击【开始】

### 应用示例 - 自动登录
```json
[
  {"type": 1.0, "value": "template/login_button.png", "retry": -1},
  {"type": 4.0, "value": "用户名", "retry": 1},
  {"type": 5.0, "value": "0.3", "retry": 1},
  {"type": 1.0, "value": "template/password_field.png", "retry": 1},
  {"type": 4.0, "value": "密码", "retry": 1},
  {"type": 1.0, "value": "template/submit_btn.png", "retry": -1}
]
```

**配置文件格式说明**：
- `type`：操作编码（1.0-9.0）
- `value`：参数（图片路径、文本、秒数等）
- `retry`：重试策略（1/n/>-1）

## ⚠️ 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 未找到匹配图片 | 图像清晰度不足、窗口遮挡或路径错误 | 重新截取清晰图像，确保app在前台，检查路径 |
| 操作超时 | 系统响应慢、网络延迟或超时设置短 | 增加等待时间（操作5.0），优化系统或网络 |
| Ctrl+V粘贴无反应 | 未用Win+Shift+S截图或输入框无焦点 | 用系统截图工具，点击参数框确保焦点 |
| 图片压缩后识别失败 | 原始图片质量差或压缩过度 | 取消压缩，重新截取高清图片 |
