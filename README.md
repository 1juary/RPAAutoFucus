# WaterRPA - 傻瓜式RPA自动化工具

**无需编程，图形化配置和执行自动化任务**

---

## 🚀 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 启动应用
python waterRPA_GUI.py
```

## 💡 功能特性

*   **9种操作类型**：包括点击、双击、右键、文本输入、等待、滚动、按键、悬停、截图。
*   **智能截图粘贴 (Ctrl+V)**：支持系统截图直接粘贴，自动保存至 `template/`，支持图片压缩。
*   **重试与循环**：灵活设置重试次数及任务循环模式。
*   **配置导入导出**：支持 JSON 格式配置分发。
*   **多线程运行**：UI 不卡顿，支持运行中一键停止及最小化。

---

## 📖 使用指南

1.  **添加指令**：点击 `+新增` 指令。
2.  **设置参数**：
    *   **图像类**：`Win+Shift+S` 截图后在输入框 `Ctrl+V`。
    *   **文本/等待**：直接输入数值或字符。
3.  **运行**：点击 `▶ 开始运行`（配合循环或最小化选项）。

---

## 🛠️ 技术栈

*   **GUI**: PySide6
*   **自动化**: PyAutoGUI, Pyperclip
*   **图像**: Pillow, OpenCV
